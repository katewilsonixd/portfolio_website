Creating a Brand.

Creating my own personal brand has been key in all the projects I have been involved in so far, especially creating a Portfolio Website.

Skills: UI, Wireframing, Sketching, Branding


The Brief.

To create my own personal brand, including a monogram, colour schemes, and a set of Brand Guidelines.

Discover.

I thought about different ways which I could create a monogram design, starting on paper.

Define and Develop.

I narrowed it down to a few basic designs. This lead me to a refined sketch.


Deliver.

I created the Monogram design digitally, displaying it in different formats such as buisness-cards.



The final product was a set of brand guidelines which reflect myself as a designer and a brand.
